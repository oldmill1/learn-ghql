Example 1: Display a value

Get a value from GraphQL and display it. Here, we're getting name from a Tour Dossier ("Kerala Beaches & Backwaters") and displaying it using a div tag.

GraphQL Response: 

      name: 'Kerala Beaches & Backwaters'

See the Pen GHQL Example 1 by Ankur Taxali (@ankurtaxali) on CodePen.

Example 2: Display a nested value

Sometimes the response will be nested inside an object. You can display it using the same way as you're used to.

GraphQL Response: 

      startAddress {
        city: 'Kochi'
      }

See the Pen GHQL Demo by Ankur Taxali (@ankurtaxali) on CodePen.

Example 3: Customize Rendering

You can control where to display the response inside the div tag by using the {value} keyword. In this example, we can capitalize the start city by wrapping it in a span tag.

Example 5: Filtering And Arrays

In this example, we want to get the Canadian price from all the prices of the room who's name contains the word Standard.

      advertisedDeparture: {
        id: 20902,
          rooms: [
            {
              code: 8 - 6,
              name: Standard - Double,
              prices: [
                {
                  currency: CAD,
                  originalPrice: 1383.20,
                  price: 1383.20
                },
                ...
